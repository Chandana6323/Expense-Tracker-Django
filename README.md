# Expense Tracker using Django

A web-based Expense Tracker application built using Django and SQLite for managing and tracking daily expenses.

## Features

- Add Expenses
- View Expenses
- Delete Expenses
- Track Total Expenses
- SQLite Database Integration
- Django Forms and Models
- User-Friendly Interface

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS

## Installation

### Clone the Repository

```bash
git clone https://github.com/Chandana6323/Expense-Tracker-Django.git
```

### Navigate to Project Folder

```bash
cd Expense-Tracker-Django
```

### Install Django

```bash
pip install django
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Server

```bash
python manage.py runserver
```

### Open in Browser

```text
http://127.0.0.1:8000/
```
